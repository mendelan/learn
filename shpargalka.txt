python -m venv venv

активує(включає)віртуал оточення
.\venv\Scripts\Activate

показує список всіх устанновлених бібліотек
pip list

установлює бібліотеки з файла
pip install -r requirements.txt

показує список усіх віток
git branch

віддає код в репозиторій на вітку main з компа
git push origin main

скачує код з репозиторію з вітки main на комп
git pull origin main

підготовлює всі файли і папки до коміту
git add * 

створює коміт(зберігає поточний стан коду)
git commit -m

git clone ссилка
